# otcuba module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otcuba\examples\otcuba

Installation
  Silent install can be done with the switch /S (e.g. openturns-otcuba-0.0-py27.exe /S).
  To test otcuba module installation, in an OpenTURNS python console type:
    import otcuba
    print otcuba.__version__

Uninstallation
  OTCuba module will be automatically uninstalled at OpenTURNS removal.
  OTCuba module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
